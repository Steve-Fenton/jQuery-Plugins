Slide Puzzle Summary

The jQuery Slide Puzzle project is a plugin that creates a slide puzzle game out of any picture, such as this shot I took of a hoverfly!

Options

classmodifier
Used to apply class names to the elements created by the plugin. You only need to specify a class modifier if you have a name-clash.
size
The number of segments across and down (i.e. 3 would be 3 segments across and 3 segments down).
complete
Script to excute on completion of a puzzle.
