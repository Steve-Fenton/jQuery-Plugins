Content Drag Scroller Example

Content Drag Scroller Summary

The jQuery Content Drag Scroller project is a plugin that adds drag-scroll behaviour to an area of your web page, like people experience when viewing PDF documents, or on a touch-screen device.

Example

Scroll The Area Below By Dragging It - this example has CSS rules to stop the scrollbars from showing, but if you switch scrollbars on the plugin plays nicely and leaving the scrollbars on also enables mouse-wheel scrolling too!

Options

classmodifier
Used to apply class names to the elements created by the plugin. You only need to specify a class modifier if you have a name-clash.
easing
How to ease the animations, recommend "easeOutExpo", which requires jQuery Easing plugin. Uses "linear" easing by default.
velocityFactor
Adjustment for the flow after the drag is ended, affects the speed and distance of the run-over.
velocityCheckWait
The intervals at which velocity is checked.
afterScrollDuration
The length of time to continue the scroll after the drag ends.
cursor
The cursor to display over the scrollable area.
width
The width for the area, default "auto"
height
The height for the area, default "400px"
